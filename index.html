HTML5

HTML5 é a versão mais recente da linguagem de marcação de hipertexto (HTML), utilizada para criar páginas web. Ele traz diversas melhorias em relação às versões anteriores, como novas tags semânticas, suporte a multimídia e APIs para desenvolvimento de aplicações web mais ricas e interativas.

Caracteristicas

  • É uma linguagem de marcação
  • É uma linguagem interpretada
  • Roda no lado do cliente
  • Não é case-sensitive
  • É formada por tags

CSS

É uma linguagem de estilo usada para descrever a apresentação de documentos HTML. Simplificando, o CSS é responsável pela aparência de um site, determinando como cores, fontes, layout e outros elementos visuais são exibidos.

Caracteristicas

  • É uma linguagem de formatação e utilização visual
  • É uma linguagem interpretada
  • Roda no lado do cliente
  • Não é case-sensitive
  • É formada por seletores
